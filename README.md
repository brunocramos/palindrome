# Simple palindrome API and Web clients

## Server
```
    cd server
    cp .env.default .env // define variables
    npm install
    // Testing:
    npm test

    // Running
    npm start
```

Total time to develop: 23min

----

## Client
```
    cd client
    npm install

    // Running
    npm start
    
    // Building
    npm run build
    http-server dist // or http-server of choice
```

Total time to develop: 41min
